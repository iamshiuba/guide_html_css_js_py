HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar páginas na web. 
Ela é usada para dar estrutura e significado ao conteúdo de uma página web.

Uma página HTML é composta por uma série de elementos, que são representados por tags, que são palavras-chave 
rodeadas por colchetes angulares (<>) e que possuem um nome e, em alguns casos, atributos.

Os elementos HTML possuem uma estrutura em forma de árvore, onde cada elemento pode ter filhos e pais. 
O elemento mais alto da árvore é o elemento <html>, que é o elemento raiz de toda a página.

Os elementos HTML podem ser divididos em dois grupos: elementos de bloco e elementos de linha. 
Os elementos de bloco ocupam toda a largura da página e são utilizados para criar seções e divisões na página. 
Os elementos de linha, por outro lado, ocupam apenas o espaço necessário para o seu conteúdo e são utilizados 
para criar parágrafos, títulos, imagens, etc.

Abaixo, vamos ver alguns exemplos de elementos HTML comuns:

<p> - parágrafo: um elemento de linha que contém um parágrafo de texto.
<h1>, <h2>, <h3>, etc. - títulos: elementos de linha que contêm títulos em diferentes níveis de importância.
<img> - imagem: um elemento de linha que contém uma imagem.
<a> - link: um elemento de linha que contém um link para outra página.
<ul>, <ol>, <li> - listas: elementos de bloco que contêm listas de itens.
<div> - divisão: um elemento de bloco que pode ser utilizado para agrupar outros elementos.
<span> - span: um elemento de linha
<form> - formulário: um elemento de bloco que pode ser utilizado para criar formulários.
<input>, <textarea>, <select> - campos de formulário: elementos de linha que podem ser utilizados 
    para criar campos de formulário.
<button> - botão: um elemento de linha que pode ser utilizado para criar botões.
<label> - rótulo: um elemento de linha que pode ser utilizado para criar rótulos para campos de formulário.
<header>, <footer>, <nav>, <main>, <section>, <article>, <aside> 
    - estrutura da página: elementos de bloco que podem ser utilizados para criar a estrutura da página.
