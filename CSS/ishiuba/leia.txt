 CSS é uma linguagem de estilo para documentos HTML e XML.
 Ela permite controlar a apresentação de um documento, como a cor, o tamanho da fonte, etc.
 A linguagem CSS é composta por seletores, propriedades e valores.

 Seletores
 Os seletores são utilizados para definir quais elementos HTML devem ser estilizados.
 Os seletores podem ser baseados em nomes de tags, classes, ids, etc.
 Exemplo: h1, .title, #header

 Propriedades
 As propriedades são utilizadas para definir quais características do elemento HTML devem ser estilizados.
 As propriedades podem ser de cor, tamanho, fonte, etc.
 Exemplo: color, font-size, font-family

 Valores
 Os valores são utilizados para definir o valor da propriedade.
 Os valores podem ser de cor, tamanho, etc.
 Exemplo: #000000, 16px, Arial
